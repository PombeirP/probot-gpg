# Deploying

If you would like to run your own instance of this plugin, see [Probot's docs for deploying plugins](https://github.com/probot/probot/blob/master/docs/deployment.md).

This plugin requires these **Permissions & events** for the GitHub App:

- Commit statuses - **Read & write**
- Pull requests - **Read-only**
  - [x] Check the box for **Pull request** events
- Repository contents - **Read-only**
  - [x] Check the box for **Push** events
